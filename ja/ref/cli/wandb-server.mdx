**使用方法**

`wandb server [OPTIONS] COMMAND [ARGS]...`

**概要**

ローカル W&B サーバーを操作するためのコマンド


**オプション**

| **オプション** | **説明** |
| :--- | :--- |


**コマンド**

| **コマンド** | **説明** |
| :--- | :--- |
| start | ローカル W&B サーバーを開始する |
| stop | ローカル W&B サーバーを停止する |
